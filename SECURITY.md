# Security Notes

This is a local portfolio demo with synthetic data only.

- No production credentials are required.
- No external API calls are made.
- No customer, vendor, employee, email, phone, token, key, or password data should be committed.
- `.env`, `.env.*`, logs, temporary screenshots, build outputs, and dependencies are excluded by `.gitignore`.

If this demo is extended with real integrations, add environment variables through a private deployment platform and keep example values in `.env.example` only.
